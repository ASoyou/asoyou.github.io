图是一种数据结构，它是一种非线性的数据结构，能用它解决的题目很多，它衍生出的数据结构：二叉树也是出题人常考的一种数据结构.

图的认识

图的构成

众所周知，图的构成就是边和节点.

节点是图的“元素”，是图的重点.

边一般是连接两个节点的线，它分无向边和有向边。无向边是没有方向的；而有向边是有方向的，它往往代表一个节点向另一个节点的流通关系。特别地，如果两个点 a 和 b，有两条边：a→b 和 b→a，那么，就认为是这两条边之间存在双向边.

特殊的图
环

在无向图上，有 n 个（n≥2）个结点，给它某一个排列，第 1 个与第 2 个存在相连的边，第 i 个与第 i+1 个存在相连的边（1≤i<n），第 n 个与第 1 个存在相连的边，如果以上条件均满足，就说明这个无向图存在环。如图：

在这张图里，1,2,5,6 就组成了一个环，即这张图就是无向图.

在有向图上，有 n 个（n≥2）个结点，给它某一个排列，存在 i→i+1 这条边（1≤i<n），存在 n→1，如果以上条件均满足，就说明这个有向图存在环。如图：

链

链是指 n 个节点（n≥2），i 和 i+1 有边（1≤i<n）.

树

树是一张没有环的无向图，树的应用非常广泛，其中，二叉树是它的代表.

拓扑

拓扑是一种基于有向图的遍历方式，它使用于无环的有向图，它用于需要合并解决很多组依赖条件，将这些条件合并成一个大条件.

拓扑+Vector

#include<bits/stdc++.h>
using namespace std;
int n , m , rd[100005] ;
vector<int> tree[100005] ;
queue<int> q ;
signed main(){
    cin >> n >> m ;
    for( int i = 1 ; i <= m ; i ++ ){
        int u , v ;
        cin >> u >> v ;
        tree[u].push_back( v ) ;
        rd[v] ++ ;
    }
    for( int i = 1 ; i <= n ; i ++ ){
        if( !rd[i] ){
            q.push( i ) ;
        }
    }
    while( !q.empty() ){
        int x = q.front() ;
        q.pop() ;
        cout << x << " " ;

图的最短路

Floyd 算法

Floyd 的核心思想是 dp 它的状态设计，它的思路十分简单明了，跑一遍 Floyd 即可求出任何一个点到另外一点的最短路.

Floyd的状态设计

fi,j:从 i 点走到 j 点的最短路.

Floyd 的操作（转移方程）

连接两条路径：

fi,j=fi,k+fk,j

Floyd 的效率

O(n3)，足够快了.

代码实现
xxxxxxxxxx
#include<bits/stdc++.h>
using namespace std;
int n , m, f[1005][1005] ;
signed main(){
    cin >> n >> m ;
    for( int i = 1 ; i <= m ; i ++ ){
        int u , v , w ;
        cin >> u >> v >> w ;
        f[u][v] = w ;
    }
    for( int k = 1 ; k <= n ; k ++ ){
        for( int i = 1 ; i <= n ; i ++ ){
            for( int j = 1 ; j <= n ; j ++ ){
                f[i][j] = max( f[i][j] , f[i][k] + f[k][j] ) ;
            }
        }
    }
    cout << f[1][n] ;
    return 0 ;
}
Floyd 的变形题

在 14 行中，我们进行了 max 和 + 操作.

在变形题中，我们会用到其他运算符获得题目要求的答案，那么我们就要修改第 14 行.

P2888 目标是求最小值，核心代码为f[i][j] = min( f[i][j] , max( f[i][k] , f[k][j] ).

P3611 目标是求或运算，核心代码为f[i][j] |= f[i][k] & f[k][j] ;.

由此，我们发现，第 14 行的操作其实是目标的反运算.

Dijkstra 算法

Dijkstra 是一种单源最短路的算法，复杂度为 O(n2)（这里的常数比较低）.

原理

Dijkstra 有以下两种操作.

选取最小

从未处理的优先队列中取出 dis 值最小的一个，将它记作 x.

松弛邻边

对 x 的所有还在未处理队列中的邻边 y，将它的 disy=min(disy,disx+tisi,j).

我们可以使用一个标记数组标记哪些数已经离开了优先队列. disi,j 表示 i→j 的边的权值.

并查集

并查集是个好用的数据结构，它其实就是一种树.

我们可以把并查集想象成是一些集团，最开始，所有人属于自己的集团，然后各个集团开始收购其他集团，被收购集团的所有人都归属新的集团.

基本操作

建树

赋值 ai=i.

find()

find 用来查找一个元素所属的集团.

查找时我们会把每一层获得的结果复制给 ai，这个操作叫路径压缩，能节省时间.

收购

收购的思路有点坑.

如果 x 要把 y 收购，就是要 y 的集团全部修改到 x 上.

所以我们要把 afind(y)=afind(x).

查询

查询也不要直接 ax，应使用 find(x).

常用性

非常常用，看起来很慢，其实复杂度蛮可观的.

最小生成树

最小生成树算法是指将一张连通图去掉若干个边使其变成一棵树.

复杂度

O(m)

核心思路

最小生成树算法是基于贪心的.

它的本质就是将 m 条边，按权值递增排序，接下来，使用并查集判断每一条边的两个节点是否不存在同一个集团，如果不是，就使用这条边，将两个节点合并到一个集团.

网络流问题

网络流问题一般指在每条边都有各自的 流量 时，从 s 点（源点）到 t 点（汇点）的几条路径的问题.

最大流

最大流是最基础的网络流，它有很多种方法解决.

最大流的术语

增广路：指在网络流中的一条从源点到汇点全程最小流量大于 0 的路径.

Dinic
算法思路

Dinic 是一种增广路最大流算法，它的原理是先用 bfs 搜索到一条增广路，再用 dfs 顺着 bfs 做的标记将沿途中的流量正向减反向加，然后再找增广路并操作，直到找不到增广路.

#include<bits/stdc++.h>
#define int long long
using namespace std;


const int INF = LLONG_MAX ;


struct Edge{
    int to , w , lip ;
} gra[1000006] ;
int n , m , s , t , head[1000006] , cnt = 1 , ans , hight[1000006] , now[1000006] ;


void csh(){
    for( int i = 1 ; i <= 200 ; i ++ ){
        hight[i] = INF ;
    }
}


void add( int a , int b , int c ){
    gra[++ cnt].to = b ;
    gra[cnt].lip = head[a] ;
    head[a] = cnt ;
    gra[cnt].w = c ;
}


bool bfs(){
    csh() ;
    queue<int> q ;
    q.push( s ) ;
    hight[s] = 0 ;
    while( !q.empty() ){
        int u = q.front() ;
        q.pop() ;
        now[u] = head[u] ;
        for( int i = head[u] ; i != 0 ; i = gra[i].lip ){
            int v = gra[i].to ;
            if( gra[i].w > 0 && hight[v] == INF ){
                q.push( v ) ;
                hight[v] = hight[u] + 1 ;
                if( v == t ) return true ;
            }
        }
    }
    return false ;
}


int dfs( int u , int sum ){
    if( u == t || sum <= 0 ){
        return sum ;
    }
    int flow = 0 ;
    for( int i = now[u] ; i != 0 ; i = gra[i].lip ){
        now[u] = i ;
        int v = gra[i].to ;
        if( gra[i].w > 0 && ( hight[u] == hight[v] - 1 ) ){
            int tmp = dfs( v , min( sum , gra[i].w ) ) ;
            gra[i].w -= tmp ;
            gra[i^1].w += tmp ;
            flow += tmp ;
            sum -= tmp ;
            if( tmp ) return flow ;
        }
    }
    return 0 ;
}


signed main(){
    scanf( "%lld %lld %lld %lld" , &n , &m , &s , &t ) ;
    for( int i = 1 ; i <= m ; i ++ ){
        int u , v , w ;
        scanf( "%lld %lld %lld" , &u , &v , &w ) ;
        add( u , v , w ) ;
        add( v , u , 0 ) ;
    }
    while( bfs() ) ans += dfs( s , INF ) ;
    printf( "%lld" , ans ) ;
    return 0 ;
}

EK 算法

算法思路

EK 算法优化了 Dinic 的 dfs 操作，在 bfs 中，将 v 入栈时，会标记 last[v] = i，这样就可以把 dfs 操作直接变成一个回溯的过程，减少了很多无用的递归.

附 EK 模板，用于通过洛谷 P3376.

xxxxxxxxxx
#include <bits/stdc++.h>
#define int long long 
using namespace std;


struct Edge{
    int w , from , to , lip ;
} gra[10005] ;
int n , m , s , t , cnt = 1 , head[205] , last[205] , ccf[205] , ans ;
void add( int a , int b , int c ){
    gra[++ cnt].lip = head[a] ;
    head[a] = cnt ;
    gra[cnt].w = c ;
    gra[cnt].from = a ;
    gra[cnt].to = b ;
}


bool bfs(){
    memset( ccf , 0 , sizeof ccf ) ; 
    queue<int> q ;
    q.push( s ) ;
    ccf[s] = 0x3f3f3f3f ;
    while( !q.empty() ){
        int u = q.front() ;
        q.pop() ;
        if( u == t ) return true ;
        for( register int i = head[u] ; i != 0 ; i = gra[i].lip ){
            int v = gra[i].to ;
            if( gra[i].w > 0 && ccf[v] == 0 && v != s ){
                ccf[v] = min( ccf[u] , gra[i].w ) ;
                last[v] = i ;
                q.push( v ) ; 
            }
        }
    }
    return false ;
}
void work(){
    while( bfs() ){
        ans += ccf[t] ;
        for( register int i = last[t] ; i != 0 ; i = last[gra[i].from] ){
            gra[i].w -= ccf[t] ;
            gra[i ^ 1].w += ccf[t] ;
        }   
    }
} 


signed main(){
    ios::sync_with_stdio(0) ;
    cin.tie(0);
    cout.tie(0);
    cin >> n >> m >> s >> t ;
    for( int i = 1 ; i <= m ; i ++ ){
        int u , v , w ;
        cin >> u >> v >> w ;
        add( u , v , w ) ;
        add( v , u , 0 ) ;
    }
    work() ;
    cout << ans ;
    return 0;
}
